# Portfolio
Personal Portfolio Website

## Deployment

This portfolio is deployed with Vercel and connected to the GitHub repository:

`github/Zachary-Levesque/Portfolio`

Production updates are automatic. Push changes to the `main` branch, and Vercel will build and publish the latest version.

Current Vercel production domain:

`portfolio-weld-sigma-65.vercel.app`

The latest production deployment can also be viewed from the Vercel project dashboard under **Overview** or **Deployments**.

## Local Development

Run the development server:

```bash
npm run dev
```

Then open:

`http://localhost:3000`
